# chat-client
一个WebRTC和WebSocket的综合例子项目，包含如下例子：
1. 基于vue、vue-socket.io的匿名聊天系统
2. WebRTC本地一对一视频通话示例
3. WebRTC本地在线一对一视频通话示例
4. 基于peerjs的WebRTC示例
5. 基于WebRTC+MediaRecorder实现录音的实例
6. 基于WebRTC+WebAUdio实现录音的例子
7. 基于WebRTC+WebAUdio实现录音压缩的例子
